# Data & Artificial Intelligence

| # | Role | Description | What They Do | Key Skills | Reports To |
|---|------|-------------|--------------|------------|------------|
| 1 | Data Engineer | Designs and builds infrastructure for collecting, processing and storing data | Build and maintain data pipelines and ETL/ELT workflows, design data lake and warehouse architectures, orchestrate batch and streaming data processing, ensure data reliability and pipeline monitoring | Python, SQL, Spark, Airflow, ETL, dbt, data lakes, Kafka, SSIS, Talend, Informatica, data pipelines | Data Architect / Engineering Manager |
| 2 | Data Scientist | Applies statistical and machine learning models to solve complex problems | Develop predictive and prescriptive models, perform exploratory data analysis and hypothesis testing, design experiments and A/B tests, translate business questions into data-driven solutions, communicate findings through statistical reports | Python, R, statistics, scikit-learn, Jupyter, pandas, hypothesis testing | CDO / Engineering Manager |
| 3 | Data Analyst | Analyzes business data to extract actionable insights and support strategic decisions | Build interactive dashboards and self-service reports, define and track KPIs and business metrics, perform ad-hoc data analysis for stakeholders, create data visualizations that drive decision-making, develop recurring reporting workflows | SQL, Excel, Tableau, Power BI, data visualization, reporting, DAX, data modeling, KPI tracking | Data Scientist / CDO |

| 5 | Data Architect | Designs the overall structure of an organization's data systems and governance policies | Define enterprise data models and schema standards, establish data governance policies and compliance frameworks, manage master data and metadata catalogs, design data quality rules and validation processes, plan data lineage and lifecycle management | Data modeling, schemas, data governance, MDM, data quality, warehousing, data catalogs, lineage, GDPR, metadata management | CDO / CTO |

| 9 | Robotics Engineer | Develops software and algorithms for controlling robots and autonomous systems | Program robot control systems and motion planning, integrate sensor data for perception and navigation, implement SLAM and path planning algorithms, develop simulation environments for testing, build autonomous decision-making systems | ROS, Python, C++, sensors, SLAM, motion planning, computer vision | Engineering Manager / CTO |
| 10 | NLP Engineer | Specialized in natural language processing (chatbots, translation, text analysis, LLM integration) | Build conversational AI and chatbot systems, develop text classification and entity recognition pipelines, fine-tune and deploy large language models, implement machine translation and summarization, design semantic search and information extraction systems | spaCy, Hugging Face, transformers, text classification, NER, LLMs | Machine Learning Engineer / Engineering Manager |
