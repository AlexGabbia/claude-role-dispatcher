# IT Support & Administration

| # | Role | Description | Key Skills | Reports To |
|---|------|-------------|------------|------------|
| 1 | IT Service Manager | Manages IT service delivery following frameworks like ITIL | ITIL, SLA management, incident management, service catalog, ITSM tools | IT Director / CIO |
| 2 | IT Coordinator | Coordinates daily IT activities and liaises with other departments | Cross-team coordination, scheduling, reporting, vendor liaison | IT Service Manager / IT Director |
| 3 | IT Support Specialist | Resolves hardware and software technical issues for enterprise users | Troubleshooting, ticketing systems, remote support, hardware repair | IT Service Manager / IT Coordinator |
| 4 | IT Auditor | Verifies IT system compliance with regulations and corporate policies | Audit frameworks, ISO 27001, SOX, risk assessment, control testing | IT Director / CISO |
| 5 | IT Compliance Officer | Ensures the IT organization complies with applicable laws and regulations | GDPR, HIPAA, SOC 2, policy writing, compliance monitoring, gap analysis | IT Director / CISO |
| 6 | IT Asset Manager | Tracks and manages the inventory of all corporate IT resources | CMDB, asset lifecycle, depreciation, inventory tools, procurement | IT Service Manager / IT Director |
| 7 | Help Desk Technician | Provides first-level technical support to users | Ticket triage, password resets, software installs, basic troubleshooting | IT Support Specialist / IT Service Manager |
| 8 | IT Procurement Specialist | Manages purchases of hardware, software and IT services | RFP/RFQ, vendor evaluation, contract negotiation, budget tracking | IT Director / IT Asset Manager |
| 9 | NOC Technician | Monitors networks and systems from the Network Operations Center | Monitoring tools, alert triage, escalation procedures, shift operations | IT Operations Manager / Network Engineer |
| 10 | Desktop Support Technician | Configures and maintains workstations (PCs, printers, peripherals) | Windows/macOS setup, imaging, driver installation, peripheral config | IT Support Specialist / IT Service Manager |
| 11 | License Manager | Manages software licenses, ensuring compliance and cost optimization | License tracking, SAM tools, audit preparation, renewal management | IT Asset Manager / IT Director |
| 12 | Field Technician | Provides technical assistance directly at the customer's site | On-site support, hardware installation, cabling, site surveys | IT Service Manager / IT Support Specialist |
| 13 | IT Secretary / IT Assistant | Supports the IT department with administrative and organizational tasks | Scheduling, documentation, expense tracking, meeting coordination | IT Director / IT Coordinator |
