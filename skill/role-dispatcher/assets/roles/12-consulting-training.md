# Consulting & Training

| # | Role | Description | What They Do | Key Skills | Reports To |
|---|------|-------------|--------------|------------|------------|
| 1 | Enterprise Architect | Defines the overall IT architecture of an organization, aligning it with business strategy | Create architecture blueprints and roadmaps, govern architecture decisions across teams, evaluate technology standards, conduct business-IT alignment assessments, define reference architectures | TOGAF, ArchiMate, business-IT alignment, architecture governance, roadmaps | CTO / CIO |
| 2 | Solution Architect | Designs complex technical solutions that meet business requirements | Design end-to-end solution architectures, evaluate integration patterns between systems, produce architecture decision records, conduct trade-off analyses, validate technical feasibility | System design, integration patterns, cloud architecture, trade-off analysis | Enterprise Architect / CTO |
| 3 | Cloud Architect | Designs scalable, secure and cost-optimized cloud architectures | Design multi-cloud and hybrid infrastructure, plan cloud migration strategies, optimize cloud spending and resource usage, implement Well-Architected reviews, define serverless and container architectures | AWS Well-Architected, multi-cloud, serverless, cost optimization, migration | Enterprise Architect / CTO |
| 4 | Software Architect | Defines high-level architecture of software applications | Define application architecture and module boundaries, select design patterns and frameworks, write architecture decision records, design microservices and domain models, review code for architectural compliance | Design patterns, microservices, DDD, clean architecture, ADRs | Enterprise Architect / VP of Engineering |
| 5 | IT Consultant | Provides strategic and technical consulting to companies on IT technologies and processes | Conduct technology assessments and audits, develop IT strategy and transformation roadmaps, recommend tooling and platform selections, plan implementation and rollout phases, benchmark processes against industry standards | Technology assessment, strategy development, implementation planning | CTO / IT Director |
| 6 | Technical Account Manager | Manages technical relationships with clients, ensuring satisfaction and adoption | Run quarterly business reviews with clients, handle technical escalations and issue resolution, drive product adoption and onboarding, gather client feedback for product teams, coordinate cross-functional support | Client relationship, technical escalation, product adoption, QBRs | Customer Success Manager / Sales Operations Manager |
| 7 | Technical Writer | Writes technical documentation: manuals, guides, API references | Author API references and developer guides, create user manuals and onboarding docs, maintain docs-as-code pipelines, develop style guides and documentation standards, produce release notes and changelogs | Markdown, docs-as-code, API docs, user guides, style guides | Engineering Manager / Product Manager |
| 8 | Technical Trainer & Coach | Trains employees and clients on technologies, and mentors IT professionals in their technical and career growth | Design training curricula and hands-on labs, deliver live workshops and e-learning courses, conduct 1:1 coaching and skill assessments, create career growth plans and learning paths, evaluate training outcomes and certifications | Curriculum design, live training, e-learning, 1:1 coaching, career pathing, skill assessment | IT Director / Engineering Manager |
| 9 | Technology Evangelist | Promotes technologies and platforms through presentations, articles, communities and developer relations | Deliver conference talks and technical demos, write blog posts and technical articles, build and nurture developer communities, create sample apps and developer resources, manage developer advocacy programs | Public speaking, blog writing, demos, community building, developer relations, social media | CTO / CMO Digital |
| 10 | IT Researcher | Conducts research on emerging technologies to evaluate enterprise adoption | Build technology radars and trend reports, develop proof-of-concept prototypes, author white papers and benchmarking studies, evaluate emerging tools and frameworks, present research findings to leadership | Technology radar, POCs, white papers, benchmarking, trend analysis | CTO / Enterprise Architect |
