# Design & User Experience

| # | Role | Description | What They Do | Key Skills | Reports To |
|---|------|-------------|--------------|------------|------------|
| 1 | Product Designer | Combines UX and UI to design digital products from concept to launch | End-to-end product flows, design critiques and reviews, cross-functional design specs, prototype validation with stakeholders | Figma, user research, prototyping, design thinking, handoff | Product Manager / VP of Engineering |
| 2 | UX Designer | Designs user experience, studying how people interact with the product | Wireframes and user flows, sitemaps and navigation structures, taxonomy and content modeling, persona creation, usability test plans | Wireframing, user flows, personas, usability testing, Figma, sitemaps, taxonomy, content modeling, navigation patterns, card sorting | Product Designer / Product Manager |
| 3 | UI Designer | Designs the visual appearance of interfaces: colors, typography, layout | High-fidelity screen mockups, landing pages and responsive web layouts, style guides and visual specifications, icon and illustration sets | Visual design, color theory, typography, design systems, Figma, HTML/CSS basics, responsive design, landing pages, visual hierarchy | Product Designer / Product Manager |
| 4 | UX Researcher | Conducts user research to understand needs, behaviors and problems | User interviews and usability studies, survey design and analysis, A/B test plans and reports, research insight presentations, persona and journey map artifacts | Interviews, surveys, A/B testing, analytics, personas, usability labs | Product Designer / Product Manager |
| 5 | Design System Lead | Creates and maintains reusable component libraries to ensure visual consistency | Component libraries and documentation, design token definitions, contribution guidelines, adoption metrics and audits, cross-team alignment workshops | Storybook, design tokens, component APIs, Figma libraries, documentation | Product Designer / VP of Engineering |
| 6 | Accessibility Specialist | Ensures digital products are usable by people with disabilities | WCAG compliance audits, screen reader testing reports, ARIA implementation reviews, accessible color palette definitions, remediation roadmaps | WCAG, screen readers, ARIA, inclusive design, color contrast, audits | Product Designer / UX Designer |
| 7 | Motion Designer | Creates animations and transitions for digital interfaces and multimedia content | UI animation prototypes, micro-interaction specifications, state machine transition maps, Lottie animation assets, animated onboarding sequences | After Effects, Lottie, CSS animations, micro-interactions, video, state machines, prototyping, animation, transitions | Product Designer / UI Designer |
