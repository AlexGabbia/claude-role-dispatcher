# Infrastructure & Operations

| # | Role | Description | What They Do | Key Skills | Reports To |
|---|------|-------------|--------------|------------|------------|
| 1 | IT Operations Manager | Oversees daily IT operations ensuring service continuity | Defines SLAs and escalation policies, coordinates incident response across teams, manages vendor contracts and renewals, plans capacity and budget for infrastructure | ITIL, incident management, capacity planning, vendor coordination | IT Director / CTO |
| 2 | Cloud Engineer | Designs and manages infrastructure on cloud platforms (AWS, Azure, GCP) | Provisions and configures cloud resources via IaC, optimizes monthly cloud spend and reserved instances, designs multi-region architectures for high availability, implements cloud networking and security groups | AWS, Azure, GCP, IaC, Terraform, networking, cost optimization | IT Operations Manager / Cloud Architect |
| 3 | DevOps Engineer | Automates development, testing and software release processes including container orchestration and release management | Builds and maintains CI/CD pipelines end-to-end, manages Kubernetes clusters and Helm deployments, implements feature flags and rollback strategies, writes infrastructure-as-code modules, coordinates release trains and semantic versioning | Jenkins, GitHub Actions, Docker, Kubernetes, Helm, Terraform, Ansible, CI/CD, GitOps, release pipelines, feature flags, RBAC | Engineering Manager / IT Operations Manager |
| 4 | System Administrator (SysAdmin) | Manages and maintains enterprise servers, operating systems, virtualization platforms and storage systems | Patches and hardens Linux/Windows servers on schedule, manages Active Directory users and group policies, administers VMware vSphere and Hyper-V clusters with live migration, configures SAN/NAS storage and backup solutions, plans disaster recovery and RAID strategies | Linux, Windows Server, Active Directory, Bash, PowerShell, patching, VMware vSphere, Hyper-V, KVM, SAN, NAS, backup, disaster recovery, RAID | IT Operations Manager |
| 5 | Network Engineer | Designs, configures, manages and secures enterprise communication networks | Designs network topologies with routing and switching, configures firewalls and IDS/IPS to enforce segmentation, implements VPN tunnels and load balancers, manages DNS infrastructure and SDN controllers, performs network hardening and penetration response | TCP/IP, routing, switching, VPN, DNS, load balancing, Cisco, SDN, firewalls, IDS/IPS, network segmentation, hardening, Palo Alto | IT Operations Manager / CISO |
| 6 | Site Reliability Engineer (SRE) | Ensures reliability and performance of production services | Defines and monitors SLOs/SLIs/error budgets, leads incident postmortems and writes blameless reviews, runs chaos engineering experiments in production, builds automated remediation runbooks, tunes alerting thresholds to reduce noise | SLOs, SLIs, incident management, chaos engineering, monitoring, Python | Engineering Manager / IT Operations Manager |
| 7 | Database Administrator (DBA) | Manages, optimizes and secures enterprise databases | Tunes slow queries and optimizes execution plans, configures replication and failover clusters, schedules backups and validates restore procedures, manages schema migrations across environments, monitors database health and connection pools | PostgreSQL, MySQL, Oracle, MongoDB, query optimization, backup, replication | IT Operations Manager / Data Architect |
| 8 | Platform Engineer | Builds and maintains internal platforms used by development teams | Develops internal developer portals and self-service tools, designs platform APIs and golden path templates, manages service mesh configuration and observability, maintains internal Kubernetes platforms and namespaces, creates onboarding workflows for new services | Kubernetes, service mesh, developer portals, platform APIs, IDP | Engineering Manager / IT Operations Manager |
| 9 | Infrastructure Engineer | Designs and manages physical and virtual data center infrastructure including telecommunications systems | Designs data center rack layouts with power and cooling plans, manages physical cabling and fiber optic runs, administers VoIP/SIP/PBX unified communications systems, plans 5G and WAN connectivity for campus networks, oversees hardware procurement and lifecycle management | Virtualization, VMware, networking, rack design, power, cooling, VoIP, SIP, PBX, unified communications, fiber optics, 5G | IT Operations Manager / Network Engineer |
