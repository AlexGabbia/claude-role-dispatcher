# Quality & Testing

| # | Role | Description | Key Skills | Reports To |
|---|------|-------------|------------|------------|
| 1 | QA Manager | Manages the quality department and defines enterprise testing standards | Test strategy, team management, quality metrics, process improvement | VP of Engineering / Engineering Manager |
| 2 | Test Architect | Designs the overall architecture of the testing framework | Test framework design, tool selection, CI integration, test patterns | QA Manager |
| 3 | Test Lead | Coordinates the testing team and defines quality strategy | Test planning, team coordination, defect triage, release readiness | QA Manager |
| 4 | QA Automation Engineer | Develops scripts and frameworks to automate software tests | Selenium, Cypress, Playwright, page objects, CI integration, Python | Test Lead / QA Manager |
| 5 | SDET (Software Dev Engineer in Test) | Developer specialized in creating test tools and infrastructure | Test frameworks, API testing, mocking, test data generation, TypeScript | Test Lead / Engineering Manager |
| 6 | QA Engineer | Verifies software quality through manual and automated tests | Test cases, bug tracking, regression testing, Jira, test plans | Test Lead / QA Manager |
| 7 | Performance Tester | Verifies software performance under load (speed, scalability, stability) | JMeter, k6, Gatling, load testing, profiling, bottleneck analysis | Test Lead / QA Manager |
| 8 | Security Tester | Performs specific tests to identify security vulnerabilities in software | OWASP ZAP, Burp Suite, SAST, DAST, security test automation | Test Lead / CISO |
| 9 | Usability Tester | Tests products with real users to evaluate ease of use and intuitiveness | User testing, think-aloud protocol, task analysis, heuristic evaluation | Test Lead / UX Designer |
| 10 | Regression Tester | Verifies that new changes haven't introduced errors in existing software | Regression suites, smoke testing, impact analysis, automated checks | Test Lead / QA Manager |
