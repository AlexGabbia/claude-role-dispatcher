# Quality & Testing

| # | Role | Description | What They Do | Key Skills | Reports To |
|---|------|-------------|--------------|------------|------------|
| 1 | QA Manager (also: Test Lead) | Manages the quality department, defines enterprise testing standards, and coordinates the testing team | Define test strategy and quality gates, manage and mentor QA team members, triage defects and assess release readiness, establish quality metrics and KPIs, coordinate cross-team testing efforts | Test strategy, team management, quality metrics, process improvement, test planning, defect triage, release readiness | VP of Engineering / Engineering Manager |
| 2 | Test Architect | Designs the overall architecture of the testing framework | Design end-to-end test framework architecture, evaluate and select testing tools and platforms, define test patterns and reusable components, integrate test suites into CI/CD pipelines | Test framework design, tool selection, CI integration, test patterns, scalability planning | QA Manager |
| 3 | QA Automation Engineer (also: SDET) | Develops scripts, frameworks, and test infrastructure to automate software tests | Build and maintain automated test suites, create test data generators and mocking utilities, implement page object models and API test harnesses, develop custom test tooling and reporting dashboards, integrate automated checks into CI pipelines | Selenium, Cypress, Playwright, page objects, CI integration, Python, TypeScript, API testing, mocking, test data generation | QA Manager |
| 4 | QA Engineer (also: Regression Tester) | Verifies software quality through manual and automated tests, including regression validation | Write and execute detailed test cases, perform regression and smoke testing on each release, track and document bugs with reproduction steps, conduct impact analysis on code changes, maintain and update regression suites | Test cases, bug tracking, regression testing, Jira, test plans, regression suites, smoke testing, impact analysis | QA Manager |
| 5 | Performance Tester | Verifies software performance under load, measuring speed, scalability, and stability | Design and execute load and stress test scenarios, profile application response times and throughput, identify performance bottlenecks and recommend fixes, produce capacity planning reports | JMeter, k6, Gatling, load testing, profiling, bottleneck analysis, capacity planning | QA Manager |
| 6 | Security Tester | Performs specific tests to identify security vulnerabilities in software | Execute SAST and DAST scans against application code, perform penetration testing on APIs and web interfaces, validate OWASP Top 10 compliance, automate security regression checks in CI, produce vulnerability assessment reports | OWASP ZAP, Burp Suite, SAST, DAST, security test automation, penetration testing | QA Manager / CISO |
| 7 | Usability Tester | Tests products with real users to evaluate ease of use and intuitiveness | Conduct moderated and unmoderated user testing sessions, apply think-aloud protocol to capture user behavior, perform heuristic evaluations against UI guidelines, analyze task completion rates and error patterns, deliver usability findings reports with actionable recommendations | User testing, think-aloud protocol, task analysis, heuristic evaluation, usability reporting | QA Manager / UX Designer |
