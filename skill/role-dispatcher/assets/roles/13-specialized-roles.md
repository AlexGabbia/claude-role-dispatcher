# Specialized Roles

| # | Role | Description | Key Skills | Reports To |
|---|------|-------------|------------|------------|
| 1 | Digital Transformation Manager | Leads the digital transformation process of an organization | Change management, digital strategy, process reengineering, roadmaps | CIO / CEO |
| 2 | Data Protection Officer (DPO) | Ensures organizational compliance with data protection regulations (GDPR) | GDPR, privacy impact assessments, data mapping, breach notification | CEO / CIO |
| 3 | Technical Recruiter | Specialized in recruiting technical and IT profiles | Sourcing, technical screening, ATS, employer branding, compensation | VP of Engineering / HR Director |
| 4 | DevRel (Developer Relations) | Builds relationships with the developer community, creating content and support | Community building, technical content, SDKs, developer experience | CTO / CMO Digital |
| 5 | IT Vendor Manager | Manages relationships with IT service and product vendors | Contract negotiation, SLA monitoring, vendor evaluation, procurement | IT Director / CIO |
| 6 | IT Legal Advisor | Provides legal counsel on technology topics: software contracts, intellectual property | Software licensing, IP law, data privacy, SaaS agreements, compliance | CIO / IT Director |
| 7 | Accessibility Engineer | Implements technical solutions to make software accessible to everyone | WCAG, ARIA, screen reader testing, semantic HTML, a11y automation | Engineering Manager / Product Designer |
| 8 | Site Builder / Webmaster | Builds and maintains websites, often using CMS like WordPress | WordPress, Webflow, DNS, hosting, SEO basics, site maintenance | IT Director / Content Manager |
| 9 | Localization Engineer | Adapts software and content for different markets and languages | i18n frameworks, l10n tools, Unicode, RTL support, translation APIs | Engineering Manager / Product Manager |
| 10 | Open Source Maintainer | Manages and coordinates the development of open source software projects | Git, code review, community management, licensing, release management | CTO / Engineering Manager |
| 11 | Scrum Product Coach | Helps organizations improve product management in the Agile context | Product discovery, outcome-driven development, team coaching, metrics | VP of Engineering / Agile Coach |
| 12 | Quantitative Analyst (Quant) | Uses mathematical and computational models for financial analysis and algorithmic trading | Python, R, statistics, stochastic calculus, Monte Carlo, time series | CDO / CTO |
| 13 | Bioinformatics Engineer | Applies computer science to biology for genomic and biomedical data analysis | Python, R, genomics, sequencing pipelines, bioinformatics databases | Engineering Manager / CTO |
| 14 | AI Ethics Specialist | Evaluates the ethical impact of artificial intelligence systems | Bias detection, fairness metrics, AI governance, responsible AI, auditing | CDO / CTO |
| 15 | IT Sustainability Manager | Manages environmental sustainability of IT infrastructure | Green IT, carbon footprint, energy efficiency, e-waste, sustainability KPIs | CIO / IT Director |
| 16 | No-Code Automation Specialist | Automates business processes with no-code tools (Zapier, Make) | Zapier, Make, Power Automate, workflow design, API connectors | IT Director / IT Operations Manager |
| 17 | Digital Accessibility Auditor | Verifies website compliance with accessibility standards (WCAG) | WCAG audits, screen reader testing, Axe, WAVE, compliance reports | Accessibility Specialist / Product Designer |
| 18 | GIS Specialist | Develops and manages geographic information systems for spatial data analysis | ArcGIS, QGIS, PostGIS, geospatial data, mapping, spatial analysis | IT Director / Engineering Manager |
