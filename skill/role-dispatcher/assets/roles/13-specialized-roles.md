# Specialized Roles

| # | Role | Description | What They Do | Key Skills | Reports To |
|---|------|-------------|--------------|------------|------------|
| 1 | Digital Transformation Manager | Leads the digital transformation process of an organization | Define digital transformation roadmaps, redesign legacy business processes, drive technology adoption across departments, measure transformation KPIs, coordinate cross-functional change initiatives | Change management, digital strategy, process reengineering, roadmaps | CIO / CEO |
| 2 | Data Protection Officer (DPO) | Ensures organizational compliance with data protection regulations (GDPR) | Conduct privacy impact assessments, maintain data processing registries, manage breach notification procedures, advise on GDPR-compliant data flows, train staff on data protection obligations | GDPR, privacy impact assessments, data mapping, breach notification | CEO / CIO |
| 3 | Technical Recruiter | Specialized in recruiting technical and IT profiles | Source candidates through technical communities and platforms, conduct technical screening interviews, manage applicant tracking pipelines, build employer branding for engineering teams, benchmark compensation packages | Sourcing, technical screening, ATS, employer branding, compensation | VP of Engineering / HR Director |
| 4 | IT Vendor Manager | Manages relationships with IT service and product vendors | Negotiate vendor contracts and SLAs, evaluate and score vendor performance, manage procurement lifecycles, conduct vendor risk assessments, optimize vendor portfolio costs | Contract negotiation, SLA monitoring, vendor evaluation, procurement | IT Director / CIO |
| 5 | Accessibility Engineer | Builds and audits accessible software ensuring compliance with accessibility standards | Implement WCAG-compliant UI components, conduct accessibility audits with automated and manual tools, test with screen readers and assistive technologies, generate compliance reports, define organizational a11y guidelines | WCAG, ARIA, screen reader testing, semantic HTML, a11y automation, Axe, WAVE, compliance reports | Engineering Manager / Product Designer |
| 6 | Site Builder / Webmaster | Builds and maintains websites, often using CMS like WordPress | Configure and customize CMS-based websites, manage DNS and hosting infrastructure, perform routine site maintenance and updates, implement basic SEO optimizations, monitor site uptime and performance | WordPress, Webflow, DNS, hosting, SEO basics, site maintenance | IT Director / Content Manager |
| 7 | Localization Engineer | Adapts software and content for different markets and languages | Integrate i18n frameworks into codebases, configure translation management pipelines, implement RTL and Unicode support, automate locale-specific builds, validate translated UI for layout and context | i18n frameworks, l10n tools, Unicode, RTL support, translation APIs | Engineering Manager / Product Manager |
| 8 | Open Source Maintainer | Manages and coordinates the development of open source software projects | Triage issues and review community pull requests, manage release cycles and changelogs, enforce contribution guidelines and licensing, grow and nurture contributor communities, maintain CI/CD for open source repos | Git, code review, community management, licensing, release management | CTO / Engineering Manager |
| 9 | Quantitative Analyst (Quant) | Uses mathematical and computational models for financial analysis and algorithmic trading | Build pricing and risk models, develop algorithmic trading strategies, run Monte Carlo simulations, analyze time series data for financial forecasting, validate models against market data | Python, R, statistics, stochastic calculus, Monte Carlo, time series | CDO / CTO |
| 10 | Bioinformatics Engineer | Applies computer science to biology for genomic and biomedical data analysis | Build and optimize genomic sequencing pipelines, analyze large-scale biological datasets, integrate bioinformatics databases, develop visualization tools for genomic data, automate variant calling workflows | Python, R, genomics, sequencing pipelines, bioinformatics databases | Engineering Manager / CTO |
| 11 | AI Ethics Specialist | Evaluates the ethical impact of artificial intelligence systems | Audit AI models for bias and fairness, define responsible AI governance frameworks, produce ethical impact assessments, design fairness metrics and monitoring dashboards, advise teams on ethical data usage | Bias detection, fairness metrics, AI governance, responsible AI, auditing | CDO / CTO |
| 12 | GIS Specialist | Develops and manages geographic information systems for spatial data analysis | Design and maintain geospatial databases, create interactive maps and spatial visualizations, perform spatial analysis and geoprocessing, integrate GIS data into web applications, process remote sensing and satellite imagery | ArcGIS, QGIS, PostGIS, geospatial data, mapping, spatial analysis | IT Director / Engineering Manager |
