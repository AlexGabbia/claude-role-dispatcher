# Cybersecurity

| # | Role | Description | What They Do | Key Skills | Reports To |
|---|------|-------------|--------------|------------|------------|
| 1 | CISO (Chief Information Security Officer) | Executive responsible for corporate cybersecurity strategy | Define security roadmap and budget, establish governance frameworks, present risk posture to the board, drive compliance programs, align security with business objectives | Risk management, compliance frameworks, security governance, budgeting | CEO / CTO |
| 2 | Security Architect | Designs the overall security architecture of an organization | Create zero-trust architecture blueprints, perform threat modeling on new systems, define network segmentation strategies, establish security design patterns, review architecture for compliance | Zero trust, network segmentation, threat modeling, security patterns | CISO |
| 3 | Security Engineer | Implements and manages security tools, monitors threats, and defends systems | Deploy and tune SIEM/IDS/IPS/EDR platforms, triage security alerts and investigate anomalies, write detection rules and playbooks, perform vulnerability scanning and hardening, operate honeypots and deception technologies | SIEM, firewalls, IDS/IPS, WAF, endpoint protection, detection engineering, log analysis, alert triage, EDR, automation | Security Architect / CISO |
| 4 | Application Security Engineer | Ensures code and application security during development | Run SAST/DAST scans on codebases, perform secure code reviews, integrate security gates into CI/CD pipelines, track and remediate OWASP Top 10 vulnerabilities, define secure coding standards | SAST, DAST, OWASP Top 10, secure code review, DevSecOps | Security Architect / Engineering Manager |
| 5 | Identity & Access Management Specialist | Manages authentication and authorization systems | Design and implement SSO/MFA solutions, configure RBAC/ABAC policies, manage privileged access management platforms, set up identity federation across environments, audit access rights and entitlements | OAuth, SAML, OIDC, LDAP, MFA, RBAC, PAM, identity federation | Security Engineer / CISO |
| 6 | Incident Response Analyst | Manages and coordinates response to cybersecurity incidents, including digital forensics | Contain and eradicate active threats, collect and preserve digital evidence with chain of custody, perform disk and memory forensic analysis, conduct post-mortem and timeline reconstruction, produce incident reports aligned to NIST IR framework | Forensics, containment, eradication, post-mortem, NIST IR framework, disk forensics, memory analysis, chain of custody, timeline analysis | Security Engineer / CISO |
| 7 | Penetration Tester | Simulates cyberattacks and conducts advanced red team operations to identify vulnerabilities | Execute web/network/mobile penetration tests, develop custom exploits and C2 infrastructure, perform social engineering campaigns, produce detailed findings reports with remediation guidance, test evasion techniques against defensive controls | Burp Suite, Metasploit, Nmap, OSINT, exploit development, C2 frameworks, social engineering, evasion techniques | Security Engineer / CISO |
| 8 | Compliance Analyst | Verifies that systems comply with security and privacy regulations | Prepare and coordinate audit evidence packages, map controls to regulatory frameworks, perform gap analyses against standards, track remediation of audit findings, maintain compliance documentation | GDPR, ISO 27001, SOC 2, HIPAA, PCI DSS, audit preparation | CISO / IT Director |
| 9 | Threat Intelligence Analyst | Collects and analyzes information on emerging cyber threats | Map adversary TTPs to MITRE ATT&CK framework, curate and correlate IOC feeds, monitor dark web forums for organizational exposure, produce strategic and tactical threat reports, brief stakeholders on emerging threat campaigns | MITRE ATT&CK, IOCs, threat feeds, dark web monitoring, reporting | Security Engineer / CISO |
| 10 | Cryptographer | Designs and analyzes cryptographic algorithms to protect communications | Design and evaluate encryption schemes for data at rest and in transit, manage PKI infrastructure and certificate lifecycles, select and implement hashing and signing algorithms, perform cryptanalysis on existing implementations, define key management policies and rotation strategies | AES, RSA, ECC, TLS, PKI, key management, hashing algorithms | Security Architect / CISO |
