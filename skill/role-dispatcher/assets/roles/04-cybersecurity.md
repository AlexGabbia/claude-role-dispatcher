# Cybersecurity

| # | Role | Description | Key Skills | Reports To |
|---|------|-------------|------------|------------|
| 1 | CISO (Chief Information Security Officer) | Executive responsible for corporate cybersecurity strategy | Risk management, compliance frameworks, security governance, budgeting | CEO / CTO |
| 2 | Security Architect | Designs the overall security architecture of an organization | Zero trust, network segmentation, threat modeling, security patterns | CISO |
| 3 | Security Engineer | Implements and manages security tools and solutions | SIEM, firewalls, IDS/IPS, WAF, endpoint protection, automation | Security Architect / CISO |
| 4 | Cybersecurity Analyst | Monitors and analyzes threats to IT system security | Log analysis, threat detection, vulnerability scanning, risk assessment | Security Engineer / CISO |
| 5 | SOC Analyst | Works in the Security Operations Center, monitoring events in real time | SIEM, alert triage, incident detection, playbooks, shift operations | Security Engineer / CISO |
| 6 | Application Security Engineer | Ensures code and application security during development | SAST, DAST, OWASP Top 10, secure code review, DevSecOps | Security Architect / Engineering Manager |
| 7 | Identity & Access Management Specialist | Manages authentication and authorization systems | OAuth, SAML, OIDC, LDAP, MFA, RBAC, PAM, identity federation | Security Engineer / CISO |
| 8 | Incident Response Analyst | Manages and coordinates response to cybersecurity incidents | Forensics, containment, eradication, post-mortem, NIST IR framework | Security Engineer / CISO |
| 9 | Penetration Tester | Simulates cyberattacks to identify system vulnerabilities | Burp Suite, Metasploit, Nmap, OSINT, web/network/mobile pentesting | Security Engineer / CISO |
| 10 | Compliance Analyst | Verifies that systems comply with security and privacy regulations | GDPR, ISO 27001, SOC 2, HIPAA, PCI DSS, audit preparation | CISO / IT Director |
| 11 | Threat Intelligence Analyst | Collects and analyzes information on emerging cyber threats | MITRE ATT&CK, IOCs, threat feeds, dark web monitoring, reporting | Security Engineer / CISO |
| 12 | Red Team Operator | Conducts advanced simulated attacks to test organizational defenses | Social engineering, exploit development, C2 frameworks, evasion | Security Architect / CISO |
| 13 | Blue Team Analyst | Actively defends systems and improves detection capabilities | Detection engineering, SIEM rules, honeypots, hardening, EDR | Security Engineer / CISO |
| 14 | Forensic Analyst | Investigates cyber incidents, collecting digital evidence | Disk forensics, memory analysis, chain of custody, timeline analysis | Security Engineer / CISO |
| 15 | Cryptographer | Designs and analyzes cryptographic algorithms to protect communications | AES, RSA, ECC, TLS, PKI, key management, hashing algorithms | Security Architect / CISO |
