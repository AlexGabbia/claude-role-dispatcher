# Game Development

| # | Role | Description | What They Do | Key Skills | Reports To |
|---|------|-------------|--------------|------------|------------|
| 1 | Game Designer | Defines core game mechanics, rules, progression systems, player experience, multiplayer modes, and in-game economy balance | Write game design documents, prototype and iterate on mechanics, design multiplayer modes and matchmaking rules, model virtual economies and reward loops, tune difficulty and progression curves | Game design documents, balancing, prototyping, player psychology, ELO/MMR systems, economy modeling, F2P metrics | Creative Director / Lead Game Designer |
| 2 | Level Designer | Creates game levels, environments, and spatial layouts that guide player flow | Build blockouts and whiteboxes for level geometry, script environmental triggers and events, pace combat encounters and exploration beats, conduct playtests and iterate on flow, craft environmental storytelling moments | Level editors, blockouts, pacing, environmental storytelling, playtesting | Lead Level Designer / Game Designer |
| 3 | Game Programmer | Implements game logic, physics, AI behaviors, core engine systems, internal tools, and editor pipelines | Write core gameplay and engine code, develop physics simulations for collisions and ragdolls, build internal editors and asset pipeline tools, optimize frame rate and memory usage, integrate third-party SDKs and middleware | C++, C#, Unity, Unreal, game loops, ECS, PhysX, collision detection, editor scripting, asset pipelines, Python | Lead Programmer / Technical Director |
| 4 | Graphics/Rendering Engineer | Develops rendering pipelines, shaders, lighting, and visual effects systems | Author custom shaders and material systems, implement real-time lighting and shadow techniques, optimize GPU draw calls and memory budgets, integrate ray tracing and global illumination, build post-processing effect stacks | OpenGL, Vulkan, DirectX, HLSL/GLSL, ray tracing, GPU optimization | Technical Director / Lead Programmer |
| 5 | Game AI Programmer | Creates NPC behaviors, pathfinding, decision trees, and enemy AI systems | Implement behavior trees and state machines for NPCs, develop pathfinding and navigation mesh solutions, design utility-based AI decision systems, train ML agents for adaptive difficulty, debug and tune AI combat encounters | Behavior trees, A*, finite state machines, utility AI, ML agents | Lead Programmer / Technical Director |
| 6 | Gameplay Programmer | Implements player controls, combat systems, inventory, and interactive mechanics | Code player movement and input handling, build combat and damage systems, implement inventory and crafting logic, tune game feel through animation blending and feedback, wire up UI interactions to game state | Input systems, game feel, animation blending, state machines, C# | Lead Programmer / Game Designer |
| 7 | Network/Multiplayer Engineer | Builds netcode, matchmaking, server architecture, and real-time synchronization | Develop client-server and P2P networking layers, implement lag compensation and rollback netcode, build matchmaking and lobby systems, maintain dedicated server infrastructure, design anti-cheat and security measures | Client-server, P2P, lag compensation, rollback, dedicated servers | Technical Director / Lead Programmer |
| 8 | Technical Artist | Bridges art and engineering: shader authoring, optimization, and art pipeline tools | Author shaders and materials in Shader Graph, optimize LOD chains and draw call budgets, build art pipeline automation tools, set up rigging and skinning workflows, profile and fix rendering performance issues | Shader Graph, material authoring, LOD, draw call optimization, rigging | Art Director / Technical Director |
| 9 | 3D Artist/Modeler | Creates 3D models, characters, props, and environments for games | Sculpt high-poly and retopologize low-poly models, create UV layouts and bake texture maps, author PBR materials and textures, build modular environment kits, prepare assets for engine import and LOD setup | Blender, Maya, ZBrush, texturing, UV mapping, PBR materials | Art Director / Lead Artist |
| 10 | 2D Artist/Sprite Artist | Creates 2D art, sprites, textures, UI elements, and pixel art | Draw character sprites and animation frames, design tilesets and background layers, create UI icons and interface elements, produce marketing art and key visuals, maintain style consistency across assets | Photoshop, Aseprite, pixel art, sprite sheets, tileset design | Art Director / Lead Artist |
| 11 | Concept Artist | Designs visual concepts for characters, environments, and props before production | Paint character and creature concept sheets, illustrate environment mood boards and keyframes, develop style guides and color palettes, sketch prop and vehicle designs, create visual targets for production teams | Digital painting, Photoshop, visual storytelling, style guides | Art Director / Creative Director |
| 12 | Animator | Creates character animations, cutscenes, and procedural animation systems | Animate character locomotion and combat cycles, set up blend trees and state machines, clean and retarget motion capture data, rig characters with IK/FK setups, produce cinematic and cutscene sequences | Maya, Blender, motion capture, rigging, blend trees, IK/FK | Animation Director / Art Director |
| 13 | VFX Artist | Designs particle systems, explosions, magic effects, and environmental VFX | Build particle systems for combat and abilities, create flipbook and shader-driven effects, design weather and ambient environmental VFX, optimize effect draw calls and overdraw, prototype VFX timing with gameplay team | Niagara, VFX Graph, particle systems, flipbooks, shader effects | Art Director / Technical Artist |
| 14 | Sound Designer | Creates sound effects, ambient audio, and interactive audio systems for games | Record and edit foley and sound effects, implement spatial audio and occlusion systems, design interactive audio triggers and layers, build ambient soundscapes for environments, integrate audio middleware with game engine | FMOD, Wwise, Audacity, foley, spatial audio, audio middleware | Audio Director / Creative Director |
| 15 | Game Music Composer | Composes adaptive and dynamic music scores, themes, and soundtracks | Compose main themes and area-specific tracks, design adaptive music layers that respond to gameplay, orchestrate and arrange for virtual or live ensembles, mix and master final audio deliverables, implement dynamic stems in audio middleware | DAW, orchestration, adaptive music, MIDI, mixing, mastering | Audio Director / Creative Director |
| 16 | Game UI/UX Designer | Designs HUD, menus, inventory screens, and overall game user experience | Wireframe and prototype HUD and menu flows, design health bars and feedback indicators, conduct usability testing with players, create accessibility options and configurations, define interaction patterns for controllers and touch | UI frameworks, HUD design, player feedback, accessibility, prototyping | Lead Designer / Art Director |
| 17 | Narrative Designer | Writes game story, dialogue, lore, branching narratives, and world-building | Author branching dialogue trees and scripts, build lore bibles and world-building documents, design quest structures and story beats, collaborate on cinematic storyboards, manage localization of narrative content | Narrative design tools, branching dialogue, lore bibles, localization | Creative Director / Lead Designer |
| 18 | Game Producer | Manages game development schedule, budget, team coordination, milestones, and player analytics | Plan milestones and track sprint progress, manage budgets and resource allocation, analyze player behavior and retention metrics, run A/B tests and interpret engagement data, coordinate cross-discipline communication | Project management, Jira, milestone planning, risk management, scoping, SQL, analytics dashboards, cohort analysis | Studio Director / Executive Producer |
| 19 | QA Tester (Games) | Tests gameplay, finds bugs, verifies fixes, and ensures quality across platforms | Write and execute test cases for gameplay features, document bugs with reproduction steps and severity, verify fixes across platforms and configurations, perform compliance and certification testing, run regression suites before milestones | Bug reporting, reproduction steps, platform testing, compliance testing | QA Lead / Game Producer |
| 20 | Game DevOps/Live Ops Engineer | Manages game servers, live updates, deployment pipelines, and service uptime | Configure and scale game server infrastructure, build CI/CD pipelines for game builds, deploy live patches and hotfixes, manage CDN distribution for game assets, monitor uptime and set up alerting dashboards | Game server hosting, CDN, live patching, feature flags, monitoring | Technical Director / Game Producer |
